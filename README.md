## Project Index

### 1. [Connected Car Telemetry](./k8s_connected-car-telemetry/README.md)
실시간 차량 텔레메트리 데이터를 안정적으로 수집·저장·조회하기 위한 **CQRS 기반 관제 아키텍처** 프로젝트

- **기간**: 2026.02.24 ~ 2026.03.13
- **형태**: 4인 팀 프로젝트
- **담당**: 백엔드 / 데이터 파이프라인 / 인프라 일부
- **주요 기술**: Kubernetes, Kafka, CQRS, Telemetry, Monitoring

### 2. [AI·IoT Routine Management System](./ai-iot-routine-system/README.md)
IoT 센서 기반 행동 인식과 AI 피드백을 결합한 **능동형 루틴 관리 시스템** 프로젝트

- **기간**: 2025.11.11 ~ 2025.12.07
- **형태**: 1인 프로젝트
- **담당**: 기획 / 설계 / 구현 전체
- **주요 기술**: IoT, Behavior Recognition, AI Feedback, Embedded System

### 3. [Network Anomaly Detection System](./network-anomaly-detection/README.md)
네트워크 트래픽을 Flow 단위로 변환·라벨링하여 공격 탐지를 수행한 **이상 탐지 시스템** 프로젝트

- **기간**: 2025.10.10 ~ 2025.12.19
- **형태**: 4인 팀 프로젝트
- **담당**: 데이터 전처리 / 라벨링 / 탐지 기능 일부 구현
- **주요 기술**: NFStream, PCAP, Flow Generation, Labeling, Intrusion Detection
